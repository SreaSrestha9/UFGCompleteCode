Prerequisites:

From Nuget Package install lastest stable version:
	> Eyes.Selenium 
	> Microsoft.NET.Test.Sdk
	> Nunit
	> Nunit3TestAdapter
	> Selenium.Firefox.Wendriver
	> Selenium.WebDriver
	> Selenium.WebDriver.ChromeDriver
	> Selenium.WebDriver.MicrosoftDriver
	> Selenium.Support

To run the modern tests for version1:
	> Go to TestCases> ModernApproach>ModernTestsV1.cs > Run


To run the modern tests for version2:
	> Go to TestCases> ModernApproach>ModernTestsV2.cs > Run

To run the traditional tests for version1:
	> Go to TestCases> TraditionalApproach > HomeTraditoinalV1 > Run

To run the traditional tests for version2:
	> Go to TestCases> TraditionalApproach > HomeTraditoinalV2 > Run
